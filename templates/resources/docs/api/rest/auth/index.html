This is the API Authentication documentation page of the Tree Menu demonstration.

Our REST API supports multiple authentication methods to secure your API endpoints.

Authentication Methods

  • Token Authentication - Simple token-based authentication
  • JWT Authentication - JSON Web Token based authentication
  • OAuth2 - Industry-standard protocol for authorization
  • API Keys - Simple API key authentication

Implementation Examples

Here's a simple example of how to authenticate with our API using JWT:


    // Example request with JWT authentication
    fetch('https://api.example.com/data', {
        method: 'GET',
        headers: {
            'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...',
            'Content-Type': 'application/json'
        }
    })
    .then(response => response.json())
    .then(data => console.log(data));
