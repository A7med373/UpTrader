This is the API Endpoints documentation page of the Tree Menu demonstration.

Our REST API provides various endpoints for interacting with the system.

Available Endpoints

Endpoint Method Description
/api/menus GET List all available menus
/api/menus/{id} GET Get details of a specific menu
/api/menus/{id}/items GET Get all items for a specific menu
/api/items/{id} GET Get details of a specific menu item

Example Request


    // Example request to get all menus
    fetch('https://api.example.com/api/menus', {
        method: 'GET',
        headers: {
            'Authorization': 'Bearer YOUR_TOKEN',
            'Content-Type': 'application/json'
        }
    })
    .then(response => response.json())
    .then(data => console.log(data));

Notice how the menu on the left highlights the current page and expands the relevant sections.
