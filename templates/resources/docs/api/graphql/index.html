This is the GraphQL API documentation page of the Tree Menu demonstration.

Our GraphQL API provides a flexible way to query and manipulate data in our system.

GraphQL Schema

Here's a simplified version of our GraphQL schema:


    type Menu {
      id: ID!
      name: String!
      items: [MenuItem!]!
    }

    type MenuItem {
      id: ID!
      name: String!
      url: String!
      isNamedUrl: Boolean!
      parent: MenuItem
      children: [MenuItem!]!
      menu: Menu!
    }

    type Query {
      menus: [Menu!]!
      menu(id: ID!): Menu
      menuByName(name: String!): Menu
      menuItems(menuId: ID!): [MenuItem!]!
      menuItem(id: ID!): MenuItem
    }

Example Query


    query {
      menu(name: "main_menu") {
        id
        name
        items {
          id
          name
          url
          children {
            id
            name
            url
          }
        }
      }
    }

Notice how the menu on the left highlights the current page and expands the relevant sections.
