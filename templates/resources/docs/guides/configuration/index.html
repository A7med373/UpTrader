This is the configuration guide page of the Tree Menu demonstration.

Learn how to configure and customize the Tree Menu application to suit your needs.

Basic Configuration

The Tree Menu application can be configured in your Django settings.py file:


# settings.py

# Tree Menu settings
TREE_MENU = {
    'CACHE_TIME': 3600,  # Cache timeout in seconds (default: 1 hour)
    'DEFAULT_MENU_TEMPLATE': 'menu/default.html',  # Default template for rendering menus
    'ACTIVE_CLASS': 'active',  # CSS class for active menu items
    'ANCESTOR_CLASS': 'ancestor',  # CSS class for ancestors of the active item
    'SHOW_FIRST_LEVEL': True,  # Always show first level of menu items
}

Menu Templates

You can customize the appearance of menus by creating your own templates. The default template structure is:


<ul>
    {% for item in menu_items %}
    <li class="{{ item.css_classes }}">
        <a href="{{ item.url }}">{{ item.name }}</a>
        {% if item.children %}
            {% include template_name with menu_items=item.children %}
        {% endif %}
    </li>
    {% endfor %}
</ul>

Advanced Configuration

For more advanced customization, you can extend the base models and template tags:

  1. Create a custom Menu model that inherits from tree_menu.models.Menu
  2. Create a custom MenuItem model that inherits from tree_menu.models.MenuItem
  3. Create custom template tags that extend the functionality of the draw_menu tag

Notice how the menu on the left highlights the current page and expands the relevant sections.
