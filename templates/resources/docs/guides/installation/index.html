This is the installation guide page of the Tree Menu demonstration.

Follow these steps to install and set up the Tree Menu application in your Django project.

Prerequisites

  • Python 3.8 or higher
  • Django 3.2 or higher
  • pip (Python package manager)

Installation Steps

  1. Install the package using pip:

    pip install django-tree-menu

  2. Add 'tree_menu' to your INSTALLED_APPS in settings.py:

    
INSTALLED_APPS = [
    ...
    'tree_menu',
    ...
]

  3. Run migrations to create the necessary database tables:

    python manage.py migrate

  4. Include the template tag in your template:

    
{% load menu_tags %}
{% draw_menu 'main_menu' %}

Verifying Installation

To verify that the installation was successful:

  1. Access the Django admin interface
  2. Create a new menu with some menu items
  3. Add the template tag to your template
  4. View the page and check if the menu is displayed correctly

Notice how the menu on the left highlights the current page and expands the relevant sections.
