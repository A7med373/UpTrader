This is the home page of the Tree Menu demonstration.

The menu on the left demonstrates a tree-structured menu with the following features:

  • All items above the selected item are expanded
  • First level of children under the selected item is also expanded
  • Active item is highlighted based on the current URL
  • Multiple menus can be displayed on the same page
  • Only one database query is used per menu

Try clicking on different menu items to see how the menu expands and collapses.
